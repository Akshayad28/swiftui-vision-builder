# SwiftUI Vision Builder

## Overview
SwiftUI Vision Builder is a web application that transforms UI images into a ready-to-run Xcode project in seconds. By uploading your UI images, the application generates a complete SwiftUI project structure, allowing developers to quickly prototype and build applications.

## Features
- Upload multiple UI images to generate a SwiftUI project.
- Automatically creates a project structure with navigation between screens.
- Provides a preview of the generated project code.
- Allows users to download the generated project as a ZIP file.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/swiftui-vision-builder.git
   ```
2. Navigate to the project directory:
   ```
   cd swiftui-vision-builder
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application
To start the development server, run:
```
npm start
```
The application will be available at `http://localhost:3000`.

### Building for Production
To create a production build, run:
```
npm run build
```
This will generate a `build` directory with the optimized application.

## Usage
1. Upload your UI images using the image uploader.
2. Customize the project generation prompt if needed.
3. Click on the "Generate Project" button to create the SwiftUI project.
4. Preview the generated code and download the project as a ZIP file.
